# Count and Say

**Link:** https://leetcode.com/problems/count-and-say

## Problem
The count-and-say sequence is defined where each term is generated by reading the previous term digit by digit. Starting from "1", each term describes consecutive groups of identical digits. Return the nth term.

## Solution
Iteratively generate each term from the previous one. For each term, scan groups of identical consecutive digits, appending the count followed by the digit to form the next term. Repeat n-1 times starting from "1".

## Code
```cpp
string countAndSay(int n) {
    string result = "1";
    for (int i = 1; i < n; i++) {
        string next;
        int j = 0;
        while (j < result.size()) {
            char c = result[j];
            int count = 0;
            while (j < result.size() && result[j] == c) { j++; count++; }
            next += to_string(count) + c;
        }
        result = next;
    }
    return result;
}
```
