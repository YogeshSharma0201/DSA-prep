# 24 Game

**Link:** https://leetcode.com/problems/24-game/

## Problem
Given an array of 4 integers, determine if you can use all four numbers exactly once with the operations +, -, *, / (and any parenthesization) to evaluate to 24. Division is real-valued, not integer division.

## Solution
Recursively pick any two numbers from the array, combine them with all possible operations (including both a/b and b/a for division), remove the two originals and insert the result, then recurse on the reduced array. When only one number remains, check if it equals 24 within a small epsilon. This implicitly handles all parenthesizations by reducing the problem size at each step.

## Code
```cpp
class Solution {
public:
    bool res = false;
    double eps = 0.0000001;
    void solve(vector<double>& arr) {
        if(res) return;
        if(arr.size() == 1) {
            if(abs(arr[0] - 24.0) < eps) res = true;
            return;
        }

        for(int i=0; i<arr.size(); i++) {
            for(int j=0; j<i; j++) {
                double a = arr[i];
                double b = arr[j];
                vector<double> next = { double(a)*b, a+b, a-b, b-a};
                if(abs(b) > eps) next.push_back(double(a)/b);
                if(abs(a) > eps) next.push_back(double(b)/a);

                arr.erase(arr.begin()+i);
                arr.erase(arr.begin()+j);

                for(int k=0; k<next.size(); k++) {
                    arr.push_back(next[k]);
                    solve(arr);
                    arr.pop_back();
                }
                arr.insert(arr.begin()+j, b);
                arr.insert(arr.begin()+i, a);
            }
        }
    }
};
```
