# 3Sum

**Link:** https://leetcode.com/problems/3sum

## Problem
Given an integer array `nums`, return all unique triplets `[nums[i], nums[j], nums[k]]` such that `i`, `j`, and `k` are distinct indices and `nums[i] + nums[j] + nums[k] == 0`. The solution set must not contain duplicate triplets.

## Solution
Sort the array first. Fix one element `x = nums[i]` in an outer loop, then use two pointers `j` (start) and `k` (end) to find pairs that sum to `-x`. Skip duplicate values of `i` to avoid duplicate triplets. When a valid triplet is found, advance `j` past any duplicates before continuing. This runs in O(n^2) time.

## Code
```cpp
vector<vector<int>> threeSum(vector<int>& nums) {
    vector<vector<int>> ret;

    sort(nums.begin(), nums.end());

    for (int i = 0; i < nums.size(); i++) {
        if (i > 0 && nums[i] == nums[i-1]) continue;
        int x = nums[i];

        int j = i + 1, k = nums.size() - 1;

        while (j < k) {
            int y = nums[j], z = nums[k];
            if (x + y + z == 0) {
                ret.push_back({x, y, z});
                j++;
                while (j + 1 < nums.size() && nums[j] == nums[j-1]) j++;
            }
            else if (x + y + z > 0) { k--; }
            else { j++; }
        }
    }

    return ret;
}
```
